Desarrollo del Proyecto

Endpoints del Backend

Estos son los puntos de acceso (endpoints) del backend, diseñados para soportar todas las historias de usuario del proyecto.

    Historias de Usuario (Arrendador)

    Gestión de Cuentas

  • POST /api/arrendador/cuentas

    HUA.1. Creación de cuenta de arrendador.

  • PUT /api/arrendador/activar/{token}

    HUA.2. Activación de cuenta por medio de un token.

  • POST /api/arrendador/login

    HUA.3. Autenticación de usuario arrendador.

    • Gestión de Propiedades

  • GET /api/propiedades/arrendador/{id}

    HUA.4. Ver las propiedades asociadas a un arrendador específico.

  • POST /api/propiedades

    HUA.5. Crear una nueva propiedad.

  • PUT /api/propiedades/{id}

    HUA.6. Editar la información de una propiedad existente.

  • POST /api/propiedades/{id}/fotos

    HUA.7. Subir fotos para una propiedad.

  • PUT /api/propiedades/desactivar/{id}

    HUA.8. Desactivar una propiedad para que no esté disponible.

    • Gestión de Solicitudes y Calificaciones

  • GET /api/solicitudes/arriendo/arrendador/{id}

    HUA.9. Ver las solicitudes de arriendo recibidas.

  • PUT /api/solicitudes/arriendo/aceptar/{id}

    HUA.10. Aceptar una solicitud de arriendo específica.

  • PUT /api/solicitudes/arriendo/cancelar/{id}

    HUA.11. Cancelar una solicitud de arriendo específica.

  • POST /api/calificaciones/arrendatario

    HUA.12. Calificar a un arrendatario.

    Historias de Usuario (Administrador)

    Gestión de Cuentas

  • POST /api/administrador/cuentas

    HUA.13. Creación de cuenta de administrador.

  • PUT /api/administrador/activar/{token}

    HUA.14. Activación de cuenta de administrador.

  • POST /api/administrador/login

    HUA.15. Autenticación de usuario administrador.

    • Gestión de Propiedades y Arriendos

  • GET /api/propiedades/buscar

    HUA.16. Buscar propiedades basándose en ciertos criterios.

  • POST /api/arriendos/solicitar

    HUA.17. Solicitar un arriendo para una propiedad.

  • GET /api/arriendos/solicitudes/{id}

    HUA.18. Ver todas las solicitudes de arriendo hechas por un usuario.

  • POST /api/arriendos/pagar/{id}

    HUA.19. Realizar el pago de un arriendo.

    • Gestión de Calificaciones

  • POST /api/calificaciones/finca

    HUA.20. Calificar una finca.

  • POST /api/calificaciones/arrendatario

    HUA.21. Calificar a un arrendatario.
